Git

  • git init: used to start git system in your local folder
  • git add .: used to add all files into git for committing
  • git commit -m "description of the change" : used to assign a commit symbol and the description of change that happened
  • git branch -M main: To change the default master named folder to main for easy recognition by Github
  • git remote add origin git@github.com : to add or push to the remote cloud of github
  • git remote -v : To confirm the existence of the origin in the git platform
  • git push -u origin main : To push main file to the remote cloud
  • git pull origin main : To pull the main repository (folder) from the main server
  • git status: checks what branch we are currently on
  • git check out -b branch name: creates a new branch and switches to it
